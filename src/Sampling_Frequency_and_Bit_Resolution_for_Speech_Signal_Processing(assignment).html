Sampling Frequency and Bit Resolution for Speech Signal Processing

1. Write a Scilab program to convert input speech signal sampled at 44.1 kHz to 22.05 kHz. The program also should plot spectra of selected segments both from original and resampled versions. By comparing respective segment spectra, comment on the effect of changing sampling frequency.

2. Write a Scilab program to convert input speech signal sampled at 44.1 kHz to 11.025 kHz. The program also should plot spectra of selected segments both from original and resampled versions. By comparing respective segment spectra, comment on the effect of changing sampling frequency.

3. Write a Scilab program to convert input speech signal sampled at 44.1 kHz to 16 kHz and 8 kHz. The program also should plot the waveforms of all the three cases one below the other. Comment on any changes that you can observe among the three. The spectra of selected segments from original and resampled versions should also be plotted. By comparing respective segment spectra, comment on the effect of changing sampling frequency.

4. Write Scilab program to quantize input speech signal at 16 bit to 4 bit data by keeping the sampling frequency constant. Write down your observation by comparing with 16 bit data

5. Write Scilab program to quantize input speech signal at 16 bit to 2 bit data by keeping the sampling frequency constant. Write down your observation by comparing with 16 bit data.

6. Write Scilab program to quantize input speech signal at 16 bit to 1 bit data by keeping the sampling frequency constant. Write down your observation by comparing with 16 bit data.

7. Write Scilab program to quantize input speech signal at 16 bit to 12 bit data by keeping the sampling frequency constant. Write down your observation by comparing with 16 bit data.

8. Write Scilab program to quantize input speech signal at 16 bit to 6 bit data by keeping the sampling frequency constant. Write down your observation by comparing with 16 bit data.

Write a Scilab program to convert input speech signal sampled at 16 kHz and 16 bit data to 16 kHz 8 bit, 4 bit and 1 bit data. The program also should plot the waveforms of all the four cases one below the other. Comment on any changes that you can observe among the four. The spectra of selected segments from original and quantized versions should also be plotted. By comparing respective segment spectra, comment on the effect of quantization.
